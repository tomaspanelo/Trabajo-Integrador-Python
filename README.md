## Instrucciones Ejecución

Usar Python 3.11 o superior y ejecutar:

Linux:
```bash
  python3.11 -m venv venv

  source venv\bin\activate

  pip install -r requirements.txt
```
Windows:
```bash
  py -m venv venv

  source venv/Scripts/activate

  pip install -r requirements.txt
```
